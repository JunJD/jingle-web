F-05 focused exploration

围绕 Command Capsule 再细分 10 个变体

F-05 的核心是一个干净的命令入口：输入、搜索、执行。这一页只在同一个母题下调整气质：更柔、更像快捷键、更扁平、更紧凑、更亮起，还有从左或从右进入的视觉流。

Original reference

F-05 原版：命令胶囊、搜索符号、绿色唤醒点

这版作为母体保留。下面 10 个变体不是换主题，而是在“桌面效率入口”的范围里看比例、符号、状态感、方向感和一点点伙伴感哪个更耐看。

F-05ACalm Command

更柔、更像搜索入口

胶囊更宽，信号更轻，整体更像安静的 launcher 输入框。

F-05BCommand Key

更像快捷命令

把搜索圆换成加号/命令感符号，更像“按一下召唤工具”。

F-05CFlat Command

更扁平、更系统

去掉投影，轮廓和符号更硬朗。适合小图标、文档和印刷。

F-05DCompact Launcher

更紧凑、更像 app icon

胶囊收短，主体更居中，放在 dock 和应用商店里更像正式图标。

F-05EGlow Command

更亮、更像激活态

加强绿色点和周围光晕，像命令入口已经被唤醒。

F-05FCompanion Command

稍微有点伙伴感

搜索圆和绿色点被轻轻处理成“眼神”，但不加嘴，保持聪明工具的克制。

F-05GLooking Left

向左看，更像在观察

眼神偏左，中间横线继续像命令线/鼻梁，整体比正视更灵动。

F-05HLooking Right

向右看，更像准备行动

眼神偏右，像在看下一个命令目标。伙伴感更自然，不会直勾勾。

F-05ILeft Flow

视觉从左边来

左侧高光进入，胶囊向右产生厚度和阴影，像从输入走向执行。

F-05JRight Flow

视觉从右边来

右上信号成为光源，胶囊向左回应，保留不对称但更有立体秩序。
