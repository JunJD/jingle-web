Final brand direction

Jingle F-05H 定稿品牌系统

最终选择 F-05H：一个向右看的命令胶囊。它保留桌面效率工具的命令入口语义，同时有一点准备行动的伙伴感。下面展示桌面 app、Web、favicon、light / dark、flat、完整 lockup、衣服和周边状态。

Core applications

从桌面到网页的主视觉

同一个 icon 在不同载体里保持识别：深色圆角底、白色命令胶囊、右侧绿色状态点和右上唤醒弧。

Desktop app

Dock 里读成一个准备执行的命令入口。

Web logo

JingleCommand everything from one place

网页导航上更像成熟效率产品，而不是抽象品牌符号。

Favicon / small icon

小尺寸保留胶囊和绿点，细节减少但语义还在。

Light / dark

深浅背景都用同一个 app icon，避免维护双版本主标。

System forms

扁平、单色和完整 lockup

正式资产包括 full mark、flat mark、mono mark、dark lockup 和 light lockup，覆盖 UI、文档、打印和合作方物料。

Flat icon

用于低复杂度 UI、说明图和小尺寸系统组件。

Monochrome

用于单色印刷、压印、合作方白标位置。

Full lockup / dark

深色发布图、启动页和演示图主标。

Full lockup / light

官网、文档和合作页面主标。

Physical feeling

衣服和周边上的感觉

F-05H 更偏产品 icon，不是纯潮牌符号。周边上适合做小徽章、贴纸和团队物料。

Light apparel

浅色衣服上像团队徽章，干净、工具感明确。

Dark apparel

深色衣服更像桌面产品团队周边。

Stickers and tote

适合贴纸、马克杯、帆布包等轻量品牌露出。

Command companion

Jingle金狗 · 准备行动

品牌语义：不是万能宠物，而是随时待命的命令伙伴。
